# JS Lib Playground

## Run

Each folder for one web page.

- launch web server:

```bash
#  replace <folder> with the selected folder
cd <folder>
python -m http.server 8080
```

- Open browser to `http://localhost:8080`

- Reload page: F5

## JS libs

- [AgGrid](https://www.ag-grid.com/)
- [Highcharts](https://www.highcharts.com/)
